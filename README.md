# CS 匹配助手

在匹配准备阶段自动读取对局信息，展示双方选手数据与实力对比。

[**下载**](https://github.com/qianjiachun/cs-match-helper/releases)

</div>

## 功能

- 对局信息：地图、倒计时、双方阵容
- 选手数据：段位、输出、评分等
- 队伍对比：雷达图、组排识别
- AI 分析：胜率预测与赛前评估（需配置 DeepSeek 密钥）

## 支持平台

- 完美对战平台

## 使用

1. 运行 `CS匹配助手.exe`
2. 启动对战平台并匹配
3. 进入准备界面后自动展示数据

## 安全声明

- 本软件不修改游戏文件，不收集数据信息，无任何风险操作
- AI 分析功能需用户自行配置 DeepSeek 密钥，密钥仅保存在本机
